# iptv-checker-tarui

## tarui普通图标转换app图标

```bash
convert -resize "32x32>" ~/assets/icon.png  ~/Desktop/icons/32x32.png
convert -resize "256x256>" ~/icon.png  ~/Desktop/icons/128x128@2x.png
convert -resize "128x128>" ~/icon.png  ~/Desktop/icons/128x128.png
convert -resize "256x256>" ~/icon.png  ~/Desktop/icon.ico
convert -resize "1024x1024>" ~/icon.png  ~/Desktop/icon.icns
```
